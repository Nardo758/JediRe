How It Works

User has WSJ subscription
         ↓

Why LLM Instead of Rules

| Approach    | Pros                | Cons                                 |
| ----------- | ------------------- | ------------------------------------ |
| Regex/Rules | Fast, no API cost   | Breaks constantly, need 1 per source |
| LLM         | Works on ANY format | ~$0.01 per newsletter                |
