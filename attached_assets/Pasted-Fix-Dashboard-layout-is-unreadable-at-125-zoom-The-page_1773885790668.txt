Fix: Dashboard layout is unreadable at 125% zoom. The page stretches too far horizontally and text is too small. Three problems to fix:
1. Base font sizes are too small. The spec used 7-9px for most elements, which is unreadable above 100% zoom. Increase the minimum font size floor across the entire terminal:

Column headers and badges: 7px → 9px minimum
Table data and labels: 8-9px → 10px minimum
Section headers: 9-10px → 11px minimum
Body text and descriptions: 10-11px → 12px minimum
Metric values: keep large sizes (16-28px) as-is

2. The layout uses fixed pixel widths everywhere instead of flex/responsive units. The ticker bars, metrics strip, and nav bar all overflow or compress at non-100% zoom. Fix:

All horizontal layouts should use display: flex with flex-wrap: wrap or overflow-x: auto — never allow content to be clipped
The portfolio metrics strip (pipeline value, deals, assets, stage counts, alerts) should wrap to 2 rows on smaller viewports instead of forcing everything into one line
The two ticker bars (news + market data) can be collapsed into one ticker bar. Two tickers stacked is too much vertical space for information that scrolls away
DataGrid columns should use minmax() or percentage widths, not fixed pixel widths
Nav bar F-key buttons: use flex-shrink: 1 and min-width: 0 so they compress gracefully instead of overflowing

3. The bottom panel (ALERTS/NEWS/EMAIL/AGENTS/TASKS/MEDIA) is competing with the main content area for space. At 125% zoom there's almost no room left for the dashboard content. Fix:

Make the bottom panel collapsible — default to collapsed (just the tab bar visible, ~28px). Click a tab to expand to 180px. Click again to collapse.
Or: move the bottom panel to a slide-out drawer from the right edge, triggered by clicking an alert badge in the top bar

Test at 125% and 150% browser zoom after making changes. Every element should be readable and no horizontal scrollbar should appear on the main viewport.

That should get the scaling fixed without requiring a full redesign. The three issues are: font floor too low, fixed pixel widths instead of flex, and too many chrome layers eating vertical space.
