## §A-OLD (for the record — why the FIRST attempt didn't count)

### Why the prior "proof" doesn't count
The DROP was gated on an agent-admission test. The captured logs:

```
Deal A (users.subscription_tier='free', UCB='operator'):
  Commentary → automation_level gate blocked
  CashFlow   → automation_level gate blocked
Deal B (users.subscription_tier='free', no UCB → COALESCE='scout'):
  Commentary → automation_level gate blocked
  CashFlow   → tier gate blocked (scout, by design)
```

Three of four cells were blocked by the **`automation_level` gate**, which fires and the
reasoning was "automation_level blocked → therefore the tier gate passed, since tier is
checked first." That is an **inference from gate ordering, not an observation of the tier
gate admitting the user.** The tier gate never produced a visible admit decision in those
cells. S1-01 forbids exactly this substitution. The column may be safe — but this log does
not prove it.

Compounding: mid-test it was discovered the column is **`NOT NULL DEFAULT 'free'`**, not
nullable. The original bug characterization (NULL-denial) was wrong; the real production
state is `'free'` (legacy vocab) on every new user. The test was rewritten on the fly and
run once through the automation_level haze — no stop, no re-review before the irreversible DDL.

- **Live-DB-query agents (Commentary, CashFlow):** fire on `research.completed`; tier read via
  `SELECT COALESCE(ucb.subscription_tier,'scout') FROM deals d JOIN users u LEFT JOIN
  user_credit_balances ucb`. This is the authoritative repoint proof path — it reads UCB directly.
- **Payload-tier agents (Research, Supply, Zoning):** read `userTier` **from the event payload**,

  `userTier: row.tier || 'scout'`, where `row.tier` comes from **`deals.tier`** — a real column
  on the deals table, written at deal creation from the request body (`tier || 'basic'`).
  ⚠️ **This is a separate tier surface the repoint did NOT touch.** `deals.tier` is set at

  **This needs its own trace in the A9 audit** — is `deals.tier` ever reconciled with UCB, or
  is it a third source of truth? (`tier || 'basic'` even defaults to legacy vocab.)
