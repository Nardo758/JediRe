REGIMEEXPAND — SOURCECOLOR HELPER FIX

CONTEXT
Per MANDATE_CONSUMER_WIRING.md (verified outcome pending), the 
Mandate v1.3 lift exposed a rendering bug in RegimeExpand:

The sourceColor() helper does exact-key lookup against expected 
source strings. The agent now writes tiered sources like 
'tier1:t12' and 'agent:cashflow' that don't match any exact key. 
These fall through to gray and render compound labels like 
'TIER1:T12' instead of clean source pills.

The investigation identified this as Tier 1 work — the small fix 
that unlocks the visible benefit of the entire Mandate lift.
