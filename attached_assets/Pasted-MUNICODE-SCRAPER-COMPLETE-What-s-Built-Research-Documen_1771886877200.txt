💾 What Gets Extracted

For each zoning district:

• ✅ Zoning code (e.g., "R-1", "C-2")
• ✅ District name/description
• ✅ Max density (units/acre)
• ✅ Max FAR (Floor Area Ratio)
• ✅ Max height (feet + stories)
• ✅ Parking requirements (spaces/unit)
• ✅ Setbacks (front/side/rear in feet)
• ✅ Min lot size (square feet)
