Document structure
The file is one sheet with four logical sections stacked vertically:

Header metadata (rows 0–3): property name, "As Of" date (3/10/2026), reporting month (03/2026)
Unit-level detail (rows 6–2033): Current / Notice / Vacant residents
Future Residents / Applicants (rows 2034–2055): signed leases not yet moved in
Summary blocks (rows 2059–2087): occupancy rollup + charge code totals

Per-unit fields (the main table)
Each unit occupies a parent row plus N child rows (one per charge code), terminated by a Total line. Extractable fields per unit:

Unit number (e.g., 141)
Unit type code (e.g., pt_1A, pt_SA, pt_2J, pt_3A) — maps to floor plan
Unit square footage
Resident ID (anonymized token like t3014178, or VACANT)
Market rent (asking rent for that unit)
Resident deposit (security deposit held)
Other deposit
Move-in date
Lease expiration date
Move-out date (populated for Notice units)
Account balance (delinquency / credit)
Charge schedule — itemized lines per unit: rent, parking, pestctrl, trash, storage, liabins, petrent, utilreb, mtmfee, renew, empdisc, patrol, otconc, termfee, misc
Total monthly charges (sum of charge lines = actual leased rent + ancillary)

Derivable per-unit metrics
From the raw fields you can compute:

Loss-to-lease per unit = market rent − in-place rent line
Ancillary income per unit = total charges − rent line
Concession burden per unit (negative charge codes: otconc, empdisc, patrol, renew)
Lease term length = expiration − move-in
Remaining lease term = expiration − as-of date
Tenure = as-of − move-in
Status flag: Current / Notice (move-out populated) / Vacant / Future
Rent PSF = rent ÷ sqft
MTM exposure (units with mtmfee charge code)

Future Residents block (signed, not moved in) — your "leases being signed"

Higher-order analytics you can build from this single file

Unit mix table (group by unit type → count, avg sqft, avg market, avg in-place, loss-to-lease)
Trade-out analysis — by joining future residents to vacating units of same type, new-lease rent vs prior-lease rent
