LOGIC

Extract a small helper (top of file or colocated util — NOT in 
proforma-adjustment.service.ts; this is frontend display logic):

  function computeDivergenceRatio(
    broker: number | null | undefined,
    override: number | null | undefined
  ): number | null {
    if (broker == null || override == null || broker === 0) return null;
    return Math.abs(override - broker) / Math.abs(broker);
  }

Call site:
  const divergence = computeDivergenceRatio(reservesRow.broker, reservesRow.override);
  const showBadge = divergence !== null && divergence > 0.5;

Helper is intentionally generic so Phase 2 can reuse on other rows 
(PW-3 insurance, PW-7 reserves T12 layer, DS-7 management fee). Do 
NOT generalize the row rendering this PR — only Replacement Reserves.

EDGE CASES (all enforced by helper)

  broker null/undefined        → no badge
