How It Works

Example User Flow:

User: "What can I build on 1950 Piedmont Circle?"

1. User Agent receives message
2. AI recognizes: Needs zoning analysis
3. Calls tool: analyze_property_zoning("1950 Piedmont Circle NE, Atlanta, GA")
4. JediRe API: Submits task to Zoning Agent
5. Zoning Agent: Runs analysis (5-30 seconds)
6. Returns: Development potential data
7. Tool formats: Natural language summary
8. AI responds:

Based on JediRe's zoning analysis for 1950 Piedmont Circle:

The property is zoned MRC-2-C (Mixed Residential Commercial). Here's 
what you can build:

Development Capacity:
• Up to 300 units
• Maximum height: 150 feet (12 stories)  
• Parking: 1.5 spaces per unit (450 total)

Allowed Uses:
- Multifamily housing ✓
- Mixed-use development ✓
- Ground-floor retail ✓

This zoning is very favorable for multifamily development. Would you 
like me to run a financial analysis to see potential returns?
