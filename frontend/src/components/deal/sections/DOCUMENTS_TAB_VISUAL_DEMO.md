2. **Filter by "Pending Review"** - See what needs attention
3. **Use tags for quick filtering** - #critical, #phase-1, etc.
4. **Check Recent Activity** - Stay updated on team actions

### For Performance Teams
1. **Check Operations first** - Monthly reports, rent rolls
2. **Monitor Compliance category** - Ensure certifications current
3. **Review Leases regularly** - Track expirations and renewals
4. **Use search for tenant names** - Quick lease lookup

- **Keyboard shortcuts**: `Ctrl+F` for search, `G` for grid, `L` for list
- **Bulk select**: Hold Shift, click multiple cards (Future feature)
- **Quick tags**: Type `#` in search to filter by tags
- **Sort by size**: Find large files quickly
