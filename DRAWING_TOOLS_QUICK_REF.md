## âš¡ Quick Actions

### Drawing
```
1. Click tool (Marker/Line/Polygon)
2. Click on map to place/draw
3. Double-click to finish (Line/Polygon)
4. Press Escape to cancel
```

### Editing
```
1. Click "Edit" tool
2. Click shape to select
3. Drag vertices to reshape
4. Drag shape to move
5. Press Delete key to remove
```

### Styling
```
1. Click "Style" button
2. Choose preset color OR
3. Pick custom colors
4. Adjust opacity (0-100%)
5. Adjust stroke (1-10px)
6. Draw with new style
```

### Saving
```
1. Click "Save" button
2. Enter descriptive name
3. Click "Save" in modal
4. Drawing persists!
```

## ğŸ’¡ Pro Tips

1. **Zoom in** before drawing for precision
2. **Edit mode** fixes mistakes easily
3. **Save often** - drawings not saved don't persist
4. **Descriptive names** - Include date and purpose
5. **Lower opacity** - See assets underneath
6. **Export backups** - Keep copies of important work
7. **Consistent colors** - Same type = same color
